Currently Support the following

  • get/{word} -- Get the translation of a word

  • get/newWords -- Get all the new translations

  • get/countAll -- The total word number in the dictionary

  • delete/{word} -- Delete request - Delete a translation, parameter is word, DELETE

  • add -- Put Request - Add a word and translation, parameter is word and translation
  • task/start -- Spider task to read from the default file and fetch translation from google
