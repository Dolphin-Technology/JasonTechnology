Currently Support the following

  • getTranslation?word= -- Get the translation of a word

  • getNewTranslations -- Get all the new translations

  • deleteTranslation -- Delete a translation, parameter is word, DELETE

  • count -- The total word number in the dictionary

  • startTranslationSpider -- Spider task to read from the default file and fetch translation from google
  • newTranslation -- A Put Request with word and translation, parameter is word and translation, PUT method
